II. Setup
- Sau khi cài đặt môi trường Nodejs và NGINX xong.
- Chạy cd /etc/nginx/ 
- Chạy ls để xem các folder có trong Nginx.
- Kiểm tra file conf bạn sẽ biết được NGINX nhận file nào là file nạp cấu hình: vi /etc/nginx/nginx.conf (open vs vim).
- Bạn sẽ thấy nginx.conf nhận nạp các file cấu h trong modules-enabled: vs notify include /etc/nginx/modules-enabled/*.conf;
- Đến file modules-enabled và tạo một folder mới với tên web tùy ý ví dụ là nodeWeb: vi /etc/nginx/sites-enabled/nodeWeb 
- Gán text sau vao file cau hinh nay:

server {
    listen 80;
    listen [::]:80;

    index index.html server.js;

    server_name theskyfund.org www.theskyfund.org;

    location / {
            proxy_pass http://localhost:5000;
            proxy_http_version 1.1;
            proxy_set_header Upgrade $http_upgrade;
            proxy_set_header Connection 'upgrade';
            proxy_set_header Host $host;
            proxy_cache_bypass $http_upgrade;
        }
}
